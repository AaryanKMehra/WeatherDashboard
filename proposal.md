# Project Proposal: Minimalist Weather Dashboard

## Overview
A clean, high-performance weather dashboard designed for daily utility. This project focuses on integrating the **Open-Meteo API** to provide real-time weather updates and a 3-day forecast with a focus on minimalist UI/UX.

## Core Features
- **Real-time Data:** Fetching current temperature, humidity, and wind speed.
- **Forecast:** A multi-day outlook showing highs and lows.
- **Dynamic Visuals:** UI adjustments based on weather codes (e.g., clear, rainy, cloudy).
- **Responsive Design:** Optimized for both desktop and mobile viewing.

## Tech Stack
- **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid layout).
- **Functionality:** Vanilla JavaScript (ES6+) using the `fetch()` API.
- **Deployment:** GitHub Pages.

## Goals
The primary goal is to demonstrate proficiency in handling asynchronous JavaScript and mapping complex JSON data into a user-friendly interface.
