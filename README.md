# WeatherDashboard

A minimalist weather application built with Vanilla JavaScript and the Open-Meteo API.

## Live Demo
https://aaryankmehra.github.io/WeatherDashboard/

## Features
- **Current Conditions:** Instant updates for temperature and humidity.
- **3-Day Forecast:** Planning ahead with daily highs and lows.
- **Minimalist Aesthetic:** A clean, grayscale interface designed for clarity.
- **Zero-Key API:** Powered by Open-Meteo for privacy and speed.

## API Used
- **Open-Meteo API:** Used for weather data retrieval without the need for API keys.
- **Location:** Currently set to Boston, MA (latitude 42.36, longitude -71.06).

## What I Learned
Through this project, I deepened my understanding of **asynchronous JavaScript**, specifically using `fetch` to handle external data. I also practiced mapping numerical weather codes to human-readable descriptions and CSS-driven visuals.
