# Learning Log: Iteration 2

### Weather Code Mapping
I implemented a lookup object to translate the API's numerical `weather_code` into Font Awesome icons and descriptive text. For example, a code of `0` renders a sun icon and the text "Clear Sky." This was much more efficient than using a long list of `if/else` statements.

### CSS Layout
I used **Flexbox** for the 3-day forecast cards. This ensures the dashboard remains responsive and columns stay aligned. I stuck to a grayscale theme (whites and light greys) to keep the focus on the data, which aligns with my usual design preference.

### Data Handling
The multi-day forecast required accessing arrays within the `daily` object of the JSON. I used a simple `for` loop to iterate through the first three days of the array and dynamically injected the HTML into the dashboard using template literals.
